# FontChangeOnApp

## 作成意図
- 社内ツール等をリリースした際に、フォントをユーザー自身で変えたいという要望があった。
- *.xamlにFontFamilyやFontSizeを書く方法だと、ビルド時にフォントが固定されてしまう。
- 設定メニューから切り替える想定ではなく、何らかのconfigファイルを起動時に読み出して上書きすればよい。
  - よくある方法: jsonでconfigファイルを読み込む。
